El Problema y La Solución

¿Por qué el sistema actual es un problema para el pasajero?

Dependencia del efectivo, falta de cambio, inseguridad y lentitud en el abordaje.

¿Qué pasaría si el pago fuese digital?

Se agiliza el proceso, se elimina el riesgo del manejo de efectivo y se crea un registro transparente.

¿Qué información obtendríamos con una App del sistema?

Datos reales sobre afluencia de pasajeros, rutas más usadas y recaudación diaria por unidad.

¡LA SOLUCIÓN!

Enrutados Venezuela resuelve todo eso con un ecosistema de tres aplicaciones móviles conectadas entre sí en tiempo real.

El "tickete" digital escaneable

¿Cómo se protege el dinero?

Medidas de seguridad del sistema

QR Único y de Un Solo Uso

Después de ser escaneado, el tickete queda inválido permanentemente. No se puede reusar ni copiar.

Anti-Fraude

Solo Conductores Aprobados

El sistema verifica digitalmente que el conductor esté activo y autorizado antes de validar cualquier cobro.

Verificado

Saldo en el Servidor

El dinero nunca "viaja" en el teléfono — el saldo real siempre vive protegido en la base de datos segura.

Cloud Safe

Tokens de Seguridad JWT

Las contraseñas nunca se guardan — usamos tokens temporales cifrados que expiran automáticamente.

Cifrado
Conectividad

Importante: ¿Qué pasa cuando no hay internet?

Validar el QR

El sistema necesita conectarse al servidor para confirmar que el pasajero tiene saldo real disponible y que el tickete es válido.

Sin conexión: NO se puede validar

Ubicación del Conductor

Para que los pasajeros vean dónde viene el bus en el mapa, el teléfono del conductor debe enviar su posición GPS constantemente.

Sin conexión: GPS en pausa

Advertencia de Seguridad

Sin internet el sistema NO puede ser validado. Esto es intencional para evitar fraudes, duplicación de ticketes y proteger tu dinero.

Preguntas frecuentes

Lo que siempre nos preguntan

¿Funciona en cualquier teléfono Android?

¡Sí! La aplicación está optimizada para funcionar correctamente en cualquier dispositivo con Android 7.0 (Nougat) o superior.

¿Las asociaciones tienen que pagar?

Para esta etapa estudiantil, el servidor es gratuito. En una implementación real a gran escala, requeriría un plan de mantenimiento mensual.

¿Qué pasa si el conductor no tiene señal?

El sistema muestra un aviso inmediato y detiene el procesamiento de pagos hasta recuperar la conexión para asegurar la integridad de los datos.

¿Podría usarse en otras ciudades?

¡Totalmente! La arquitectura es flexible y configurable. Se pueden crear instancias separadas para diferentes asociaciones o ciudades.

¿Se puede hackear el QR?

No. Cada código QR contiene un token cifrado único que expira después de un uso o un tiempo límite. Una captura de pantalla no servirá.
